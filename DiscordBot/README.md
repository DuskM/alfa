# Discord bot.

## Commands:
- !ping (returns pong)
  
## Install.
- Install [Node.js](https://nodejs.org/en/)  
- Open a terminal and type:
- ``` npm install ```
- To use live reload type: ``` npm i -g nodemon ```
  
## How to use:
### Windows:
- Dubble click start_bot.bat

### Linux: 
- Open a terminal and type:
- ``` chmod 777 StartBot.sh ```
- ``` ./StartBot.sh ```
